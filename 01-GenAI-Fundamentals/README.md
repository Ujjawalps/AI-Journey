# 📚 Module 1 – GenAI Fundamentals

> Learn the foundations of Generative AI by building small projects from scratch.

This module introduces the core concepts required to work with Large Language Models (LLMs). Each project builds upon the previous one, so it is recommended to follow them in the given order.

---

## 🗂️ Learning Order

| # | Project | Status | Concepts |
|---|---------|--------|----------|
| 1 | ✅ WeatherApp_Ai | Completed | LLM APIs, System Prompt, Messages, Manual Tool Calling, Agent Loop |
| 2 | ✅ ChatBot | Completed | Multi-turn Conversation, Conversation Memory, Context, Message History |
| 3 | ✅ Streaming-Chat | Completed | Streaming Responses, Async Iterators, Chunks, `process.stdout.write()` |
| 4 | ⏳ JSON-Generator | Coming Next | Structured Outputs, JSON Mode, Parsing, Machine-readable Responses |
| 5 | ⏳ Email-Writer | Planned | Prompt Engineering, Tone, Templates, AI Writing |
| 6 | ⏳ Prompt-Lab | Planned | Prompt Engineering, Roles, Constraints, Few-shot Prompting |

---

## 🎯 Module Objectives

By the end of this module, I will understand:

- How LLM APIs work
- System, User, and Assistant roles
- Conversation memory
- Streaming responses
- Structured JSON outputs
- Prompt engineering fundamentals

These concepts form the foundation for the next modules on LLM Engineering and Agentic AI.
