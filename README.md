## 🎯 About This Repository

This repository documents my complete learning journey into **Generative AI**, **LLM Engineering**, **Agentic AI**, **RAG**, **Multi-Agent Systems**, and **Production AI**.

Instead of only watching tutorials, every concept I learn is implemented through a hands-on project. Each project contains:

- 📖 Detailed notes
- 🧠 Engineering concepts
- 💻 Source code
- 📝 Interview notes
- 🚀 Practical implementations

The goal is to understand **how AI systems actually work** rather than simply using them.

I believe the best way to learn AI Engineering is by building.

Every session follows three steps:

1. 📚 Learn the concept.
2. 💻 Build a working project.
3. 🧠 Understand why it works.

This repository is not just a collection of code—it's my engineering journal.
